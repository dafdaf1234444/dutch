# Everyday Dutch Phrases

A broader collection of ~80 phrases for daily life in the Netherlands.

---

## 1. Morning Routine

| Dutch | English | Pronunciation Hint |
|---|---|---|
| Goedemorgen! | Good morning! | KHOO-duh-MOR-khuh |
| Heb je goed geslapen? | Did you sleep well? | Hep yuh khoot khuh-SLAH-pun? |
| Ik ga douchen | I'm going to shower | Ik khah DOW-shun |
| Wat eten we als ontbijt? | What are we having for breakfast? | Vaht AY-tun vuh ahls ont-BAYT? |
| Ik ben laat! | I'm late! | Ik ben laht! |
| Wil je koffie? | Do you want coffee? | Vil yuh KOF-fee? |
| Ik moet me aankleden | I need to get dressed | Ik moot muh AHN-klay-dun |
| Schiet op! | Hurry up! | Skheet op! |

## 2. At Home

| Dutch | English | Pronunciation Hint |
|---|---|---|
| Ik ben thuis | I'm home | Ik ben tows |
| Wat eten we vanavond? | What are we eating tonight? | Vaht AY-tun vuh vahn-AH-vont? |
| Doe de deur dicht | Close the door | Doo duh dur dikht |
| Heb je de sleutels? | Do you have the keys? | Hep yuh duh SLUR-tuls? |
| Ik ga naar bed | I'm going to bed | Ik khah nahr bet |
| Welterusten | Good night | VEL-tuh-RUS-tun |
| Doe het licht aan/uit | Turn the light on/off | Doo ut likht ahn/owt |
| Het is tijd om op te ruimen | It's time to tidy up | Het is tayt om op tuh ROW-mun |

## 4. Socializing

| Dutch | English | Pronunciation Hint |
|---|---|---|
| Heb je zin om...? | Do you feel like...? | Hep yuh zin om...? |
| Zullen we afspreken? | Shall we meet up? | ZUL-lun vuh AHF-spray-kun? |
| Wat leuk! | How nice! / How fun! | Vaht lurk! |
| Gefeliciteerd! | Congratulations! | Khuh-fay-lee-see-TAYRT! |
| Veel plezier! | Have fun! | Vayl pluh-ZEER! |
| Ik verheug me erop | I'm looking forward to it | Ik vehr-HURKH muh ehr-OP |
| Heb je zin in een biertje? | Do you fancy a beer? | Hep yuh zin in un BEER-tyuh? |
| Dat was gezellig | That was a good time | Daht vahs khuh-ZEL-likh |
| Zullen we iets gaan drinken? | Shall we go for a drink? | ZUL-lun vuh eets khahn DRINK-un? |
| Kom je ook? | Are you coming too? | Kom yuh ohk? |

## 5. On the Phone

| Dutch | English | Pronunciation Hint |
|---|---|---|
| Met... (naam) | This is... (name) | Met... |
| Kan ik ... spreken? | Can I speak to...? | Kahn ik ... SPRAY-kun? |
| Ik bel je later terug | I'll call you back later | Ik bel yuh LAH-tur tuh-RUKH |
| Kun je dat herhalen? | Can you repeat that? | Kun yuh daht hehr-HAH-lun? |
| Je valt weg | You're breaking up | Yuh vahlt vekh |
| Ik stuur je een berichtje | I'll send you a message | Ik stuur yuh un buh-RIKHT-yuh |

## 6. Weather Talk

| Dutch | English | Pronunciation Hint |
|---|---|---|
| Wat voor weer wordt het? | What will the weather be like? | Vaht vohr vayr vort ut? |
| Het regent | It's raining | Het RAY-khunt |
| Wat een mooi weer! | What beautiful weather! | Vaht un mohy vayr! |
| Het is koud | It's cold | Het iskowt |
| Het is warm | It's warm | Het is vahrm |
| Het waait hard | It's very windy | Het vahy-t hahrt |

## 7. Opinions & Feelings
