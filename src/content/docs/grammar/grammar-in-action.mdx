## The V2 Rule

In Dutch main clauses, the conjugated verb **always** goes in position 2. The subject can move, but the verb stays put.

> <span class="g-subject">Ik</span> <span class="g-verb">drink</span> <span class="g-object">koffie</span>.
> */ɪk drɪŋk ˈkɔ.fi/*

*I drink coffee.*

> <span class="g-time">Elke ochtend</span> <span class="g-verb">drink</span> <span class="g-subject">ik</span> <span class="g-object">koffie</span>.
> */ˈɛl.kə ˈɔx.tənt drɪŋk ɪk ˈkɔ.fi/*

*Every morning I drink coffee.* — Time moved to position 1, so subject and verb swap (inversion).

> <span class="g-object">Koffie</span> <span class="g-verb">drink</span> <span class="g-subject">ik</span> <span class="g-time">elke ochtend</span>.
> */ˈkɔ.fi drɪŋk ɪk ˈɛl.kə ˈɔx.tənt/*

*Coffee I drink every morning.* — Object in position 1 for emphasis; verb still in position 2.

:::tip[The golden rule]
No matter what fills position 1, the verb is always the second element. This is the single most important rule in Dutch word order.
:::

## Time – Manner – Place

When a sentence has time, manner, and place information, they follow a fixed order after the verb.

> <span class="g-subject">Wij</span> <span class="g-verb">gaan</span> <span class="g-time">morgen</span> <span class="g-manner">met de fiets</span> <span class="g-place">naar school</span>.
> */ʋɛi̯ ɣaːn ˈmɔr.ɣən mɛt də fits naːr sxoːl/*

*We go tomorrow by bike to school.*

> <span class="g-subject">Zij</span> <span class="g-verb">rijdt</span> <span class="g-time">elke dag</span> <span class="g-manner">snel</span> <span class="g-place">naar haar werk</span>.
> */zɛi̯ rɛi̯t ˈɛl.kə dɑx snɛl naːr haːr ʋɛrk/*

*She drives every day quickly to her work.*

> <span class="g-time">Vanavond</span> <span class="g-verb">gaan</span> <span class="g-subject">we</span> <span class="g-manner">gezellig</span> <span class="g-place">naar het restaurant</span>.
> */ˌvɑnˈaː.vɔnt ɣaːn ʋə ɣəˈzɛ.ləx naːr hɛt rɛs.toːˈrɑnt/*

*Tonight we go cozily to the restaurant.* — Note "vanavond" `/ˌvɑnˈaː.vɔnt/` in position 1 triggers inversion.

## Subordinate Clauses

In subordinate clauses (after conjunctions like *dat*, *omdat*, *als*, *wanneer*), the verb moves to the **end**.

> <span class="g-subject">Ik</span> <span class="g-verb">weet</span> <span class="g-conjunction">dat</span> <span class="g-subject">hij</span> <span class="g-time">morgen</span> <span class="g-verb">komt</span>.
> */ɪk ʋeːt dɑt hɛi̯ ˈmɔr.ɣən kɔmt/*

*I know that he comes tomorrow.* — "komt" goes to the end after "dat".

> <span class="g-subject">Ze</span> <span class="g-verb">blijft</span> <span class="g-place">thuis</span> <span class="g-conjunction">omdat</span> <span class="g-subject">ze</span> <span class="g-manner">ziek</span> <span class="g-verb">is</span>.
> */zə blɛi̯ft tœy̯s ɔmˈdɑt zə zik ɪs/*

*She stays home because she is sick.* — "is" goes to the end after "omdat".

> <span class="g-conjunction">Als</span> <span class="g-subject">het</span> <span class="g-time">morgen</span> <span class="g-verb">regent</span>, <span class="g-verb">blijven</span> <span class="g-subject">we</span> <span class="g-place">thuis</span>.
> */ɑls hɛt ˈmɔr.ɣən ˈreː.ɣənt blɛi̯.vən ʋə tœy̯s/*

*If it rains tomorrow, we stay home.* — The subordinate clause fills position 1, so the main clause starts with its verb (inversion).

## Two Verbs: Modal + Infinitive

When there are two verbs (a modal and an infinitive), the modal takes position 2 and the infinitive goes to the end.

> <span class="g-subject">Ik</span> <span class="g-verb">moet</span> <span class="g-time">vandaag</span> <span class="g-manner">hard</span> <span class="g-participle">werken</span>.
> */ɪk mut vɑnˈdaːx hɑrt ˈʋɛr.kən/*

*I must work hard today.* — "moet" in position 2, "werken" at the end.

> <span class="g-subject">Zij</span> <span class="g-verb">wil</span> <span class="g-time">vanavond</span> <span class="g-object">een film</span> <span class="g-participle">kijken</span>.
> */zɛi̯ ʋɪl ˌvɑnˈaː.vɔnt ən fɪlm ˈkɛi̯.kən/*

*She wants to watch a movie tonight.*

> <span class="g-subject">Wij</span> <span class="g-verb">kunnen</span> <span class="g-place">hier</span> <span class="g-negation">niet</span> <span class="g-participle">parkeren</span>.
> */ʋɛi̯ ˈkʏ.nən hiːr nit pɑrˈkeː.rən/*

*We cannot park here.* — Negation goes before the infinitive.

## Perfect Tense

The auxiliary (hebben/zijn) takes position 2; the past participle goes to the end.

> <span class="g-subject">Ik</span> <span class="g-verb">heb</span> <span class="g-time">gisteren</span> <span class="g-object">een boek</span> <span class="g-participle">gelezen</span>.
> */ɪk hɛp ˈɣɪs.tə.rən ən buk ɣəˈleː.zən/*

*I read a book yesterday.* — "heb" in position 2, "gelezen" `/ɣəˈleː.zən/` at the end.

> <span class="g-subject">Zij</span> <span class="g-verb">is</span> <span class="g-time">vorige week</span> <span class="g-place">naar Amsterdam</span> <span class="g-participle">gereisd</span>.
> */zɛi̯ ɪs ˈvoː.rə.ɣə ʋeːk naːr ˌɑm.stərˈdɑm ɣəˈrɛi̯st/*

*She traveled to Amsterdam last week.* — Movement verbs use "zijn".

> <span class="g-time">Gisteren</span> <span class="g-verb">hebben</span> <span class="g-subject">we</span> <span class="g-manner">lekker</span> <span class="g-participle">gegeten</span>.
> */ˈɣɪs.tə.rən ˈhɛ.bən ʋə ˈlɛ.kər ɣəˈɣeː.tən/*

*Yesterday we ate well.* — Time in position 1 triggers inversion.

---

## Negation

"Niet" (not) goes as far to the right as possible, but before the final verb, preposition phrases, or adjective complements.

> <span class="g-subject">Ik</span> <span class="g-verb">drink</span> <span class="g-negation">niet</span>.
> */ɪk drɪŋk nit/*

*I don't drink.*

> <span class="g-subject">Ik</span> <span class="g-verb">drink</span> <span class="g-negation">geen</span> <span class="g-object">koffie</span>.
> */ɪk drɪŋk ɣeːn ˈkɔ.fi/*

*I don't drink coffee.* — "Geen" replaces "niet + een" before a noun.

> <span class="g-subject">Hij</span> <span class="g-verb">heeft</span> <span class="g-object">het boek</span> <span class="g-negation">niet</span> <span class="g-participle">gelezen</span>.
> */hɛi̯ heːft hɛt buk nit ɣəˈleː.zən/*

*He has not read the book.* — "Niet" before the past participle.

> <span class="g-subject">Ze</span> <span class="g-verb">gaat</span> <span class="g-negation">niet</span> <span class="g-place">naar school</span>.
> */zə ɣaːt nit naːr sxoːl/*

*She doesn't go to school.* — "Niet" before the prepositional phrase.

## Putting It All Together

Here's a paragraph that combines multiple rules. Read it slowly and notice how each rule applies:

> <span class="g-time">Gisteren</span> <span class="g-verb">ben</span> <span class="g-subject">ik</span> <span class="g-manner">met de trein</span> <span class="g-place">naar Utrecht</span> <span class="g-participle">gereisd</span>.
> */ˈɣɪs.tə.rən bɛn ɪk mɛt də trɛi̯n naːr ˈyː.trɛxt ɣəˈrɛi̯st/*

*Yesterday I traveled by train to Utrecht.* — V2 with inversion, time-manner-place, perfect tense with "zijn".

> <span class="g-subject">Ik</span> <span class="g-verb">moest</span> <span class="g-place">daar</span> <span class="g-participle">zijn</span> <span class="g-conjunction">omdat</span> <span class="g-subject">ik</span> <span class="g-object">een vergadering</span> <span class="g-verb">had</span>.
> */ɪk must daːr zɛi̯n ɔmˈdɑt ɪk ən vərˈɣaː.də.rɪŋ hɑt/*

*I had to be there because I had a meeting.* — Modal + infinitive in main clause, verb-final in subordinate clause.

> <span class="g-conjunction">Hoewel</span> <span class="g-subject">ik</span> <span class="g-manner">moe</span> <span class="g-verb">was</span>, <span class="g-verb">heb</span> <span class="g-subject">ik</span> <span class="g-manner">goed</span> <span class="g-participle">gepresenteerd</span>.
> */huˈʋɛl ɪk mu ʋɑs hɛp ɪk ɣut ɣə.preː.zənˈteːrt/*

*Although I was tired, I presented well.* — Subordinate clause first, then V2 inversion in the main clause.

> <span class="g-subject">Mijn collega's</span> <span class="g-verb">hebben</span> <span class="g-time">daarna</span> <span class="g-manner">gezegd</span> <span class="g-conjunction">dat</span> <span class="g-subject">ze</span> <span class="g-object">het</span> <span class="g-manner">heel interessant</span> <span class="g-verb">vonden</span>.
> */mɛi̯n kɔˈleː.ɣaːs ˈhɛ.bən daːrˈnaː ɣəˈzɛxt dɑt zə hɛt heːl ˌɪn.tə.rɛˈsɑnt ˈvɔn.dən/*

*My colleagues said afterwards that they found it very interesting.* — Perfect tense main clause with embedded subordinate clause.

> <span class="g-time">Volgende keer</span> <span class="g-verb">wil</span> <span class="g-subject">ik</span> <span class="g-manner">beter</span> <span class="g-participle">voorbereid</span> <span class="g-participle">zijn</span>, <span class="g-conjunction">want</span> <span class="g-subject">ik</span> <span class="g-verb">had</span> <span class="g-negation">niet</span> <span class="g-manner">genoeg</span> <span class="g-object">tijd</span>.
> */ˈvɔl.ɣən.də keːr ʋɪl ɪk ˈbeː.tər ˈvoːr.bə.rɛi̯t zɛi̯n ʋɑnt ɪk hɑt nit ɣəˈnux tɛi̯t/*

*Next time I want to be better prepared, because I didn't have enough time.* — Modal + double infinitive, "want" keeps normal word order.
