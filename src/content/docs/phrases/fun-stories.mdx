## Story 1: Het Eerste Date (The First Date)

*Level: A1 — present tense, basic word order*

| Dutch | English |
|-------|---------|
| Anna zit in een café. | Anna is sitting in a café. |
| Ze wacht op haar date. | She is waiting for her date. |
| Een man komt binnen. | A man comes in. |
| Hij draagt een oranje broek. | He is wearing orange pants. |
| "Hallo! Ik ben Mark," zegt hij. | "Hello! I am Mark," he says. |
| "Hoi Mark. Leuke broek," zegt Anna. | "Hi Mark. Nice pants," says Anna. |
| Mark lacht. "Dank je. Het is mijn geluksbroek." | Mark laughs. "Thanks. It's my lucky pants." |
| Anna bestelt koffie. Mark bestelt thee. | Anna orders coffee. Mark orders tea. |
| "Ik drink geen koffie," zegt Mark. | "I don't drink coffee," says Mark. |
| "Ik drink geen thee," zegt Anna. | "I don't drink tea," says Anna. |
| Ze kijken naar elkaar en lachen. | They look at each other and laugh. |
| Het is een goed begin. | It is a good start. |

**Key Pronunciation**

| Word | IPA |
|------|-----|
| café | `/kaˈfeː/` |
| wacht | `/ʋɑxt/` |
| broek | `/bruk/` |
| geluksbroek | `/ɣəˈlʏksbruk/` |
| bestelt | `/bəˈstɛlt/` |
| kijken | `/ˈkɛi.kən/` |

## Story 2: De Huisgenoten (The Housemates)

*Level: A1–A2 — modal verbs, negation*

| Dutch | English |
|-------|---------|
| Tom en Lisa wonen samen. | Tom and Lisa live together. |
| Tom kookt elke avond. | Tom cooks every evening. |
| Het probleem: Tom kan niet koken. | The problem: Tom cannot cook. |
| "Wat is dit?" vraagt Lisa. | "What is this?" asks Lisa. |
| "Pasta met kaas," zegt Tom trots. | "Pasta with cheese," says Tom proudly. |
| "Het is zwart," zegt Lisa. | "It is black," says Lisa. |
| "Dat is de smaak," zegt Tom. | "That is the flavor," says Tom. |
| Lisa opent het raam. De rook gaat naar buiten. | Lisa opens the window. The smoke goes outside. |
| De buren bellen aan. "Is er brand?" | The neighbors ring the bell. "Is there a fire?" |
| "Nee," zegt Lisa. "Tom kookt weer." | "No," says Lisa. "Tom is cooking again." |
| De buren brengen pizza mee. | The neighbors bring pizza. |
| Iedereen is blij. Behalve Tom. | Everyone is happy. Except Tom. |

**Key Pronunciation**

| Word | IPA |
|------|-----|
| huisgenoten | `/ˈhœys.ɣə.noː.tən/` |
| probleem | `/proːˈbleːm/` |
| zwart | `/zʋɑrt/` |
| raam | `/raːm/` |
| buren | `/ˈbyː.rən/` |

## Story 3: Bij de Dokter (At the Doctor)

*Level: A2 — past tense, compound sentences*

| Dutch | English |
|-------|---------|
| Meneer de Vries gaat naar de dokter. | Mr. de Vries goes to the doctor. |
| "Wat is het probleem?" vraagt de dokter. | "What is the problem?" asks the doctor. |
| "Ik heb overal pijn," zegt meneer de Vries. | "I have pain everywhere," says Mr. de Vries. |
| De dokter onderzoekt hem. | The doctor examines him. |
| "Doet dit pijn?" De dokter drukt op zijn arm. | "Does this hurt?" The doctor presses his arm. |
| "Ja! Au!" schreeuwt meneer de Vries. | "Yes! Ow!" screams Mr. de Vries. |
| "En dit?" De dokter drukt op zijn been. | "And this?" The doctor presses his leg. |
| "Au! Ja, ook!" | "Ow! Yes, also!" |
| De dokter denkt na. "Ik weet het al." | The doctor thinks. "I already know." |
| "Wat is het? Is het ernstig?" | "What is it? Is it serious?" |
| "Uw vinger is gebroken," zegt de dokter. | "Your finger is broken," says the doctor. |
| Meneer de Vries kijkt naar zijn vinger. "O." | Mr. de Vries looks at his finger. "Oh." |

**Key Pronunciation**

| Word | IPA |
|------|-----|
| dokter | `/ˈdɔk.tər/` |
| overal | `/oː.vəˈrɑl/` |
| onderzoekt | `/ˌɔn.dərˈzukt/` |
| schreeuwt | `/sxreːut/` |
| gebroken | `/ɣəˈbroː.kən/` |
| vinger | `/ˈvɪŋ.ər/` |

## Story 4: De Buren (The Neighbors)

*Level: A2–B1 — subordinate clauses, er + preposition*

| Dutch | English |
|-------|---------|
| Mevrouw Jansen heeft nieuwe buren. | Mrs. Jansen has new neighbors. |
| Ze vindt ze aardig, maar ze zijn luidruchtig. | She finds them nice, but they are noisy. |
| Elke vrijdagavond geven ze een feest. | Every Friday evening they throw a party. |
| De muziek is zo hard dat de muren trillen. | The music is so loud that the walls shake. |
| Mevrouw Jansen kan niet slapen. | Mrs. Jansen cannot sleep. |
| Ze belt aan bij de buren. | She rings the doorbell at the neighbors'. |
| "Het is middernacht," zegt ze vriendelijk. | "It is midnight," she says politely. |
| "Ja! Wilt u binnenkomen?" vraagt de buurman. | "Yes! Do you want to come in?" asks the neighbor. |
| Mevrouw Jansen denkt even na. | Mrs. Jansen thinks for a moment. |
| "Vooruit dan maar," zegt ze. | "Oh, all right then," she says. |
| Om drie uur 's nachts danst mevrouw Jansen op tafel. | At three in the morning Mrs. Jansen is dancing on the table. |
| De volgende vrijdag belt ze niet meer aan. | The following Friday she doesn't ring anymore. |
| Ze staat al klaar met haar dansschoenen. | She is already ready with her dancing shoes. |

**Key Pronunciation**

| Word | IPA |
|------|-----|
| luidruchtig | `/ˈlœy.drʏx.təx/` |
| middernacht | `/ˈmɪ.dər.nɑxt/` |
| vriendelijk | `/ˈvrin.də.lək/` |
| binnenkomen | `/ˈbɪ.nən.koː.mən/` |
| dansschoenen | `/ˈdɑns.sxu.nən/` |

## Story 5: Het Sollicitatiegesprek (The Job Interview)

*Level: B1 — perfect tense, formal register, complex sentences*

| Dutch | English |
|-------|---------|
| Sophie heeft een sollicitatiegesprek bij een groot bedrijf. | Sophie has a job interview at a large company. |
| Ze heeft zich goed voorbereid. | She has prepared well. |
| "Waarom wilt u hier werken?" vraagt de manager. | "Why do you want to work here?" asks the manager. |
| "Ik ben gepassioneerd over uw producten," zegt Sophie. | "I am passionate about your products," says Sophie. |
| "Welke producten maken wij?" vraagt de manager. | "Which products do we make?" asks the manager. |
| Sophie wordt rood. Ze is de naam vergeten. | Sophie turns red. She has forgotten the name. |
| "U maakt... uitstekende... dingen," zegt ze. | "You make... excellent... things," she says. |
| De manager lacht. "Dat klopt. Wij maken dingen." | The manager laughs. "That's right. We make things." |
| "Wat is uw grootste zwakte?" vraagt hij daarna. | "What is your greatest weakness?" he then asks. |
| "Ik ben te eerlijk," zegt Sophie. | "I am too honest," says Sophie. |
| "Dat is geen zwakte," zegt de manager. | "That is not a weakness," says the manager. |
| "Het kan me niet schelen wat u denkt," zegt Sophie. | "I don't care what you think," says Sophie. |
| Er valt een stilte. Dan lacht de manager hard. | There is a silence. Then the manager laughs loudly. |
| Sophie krijgt de baan. | Sophie gets the job. |

**Key Pronunciation**

| Word | IPA |
|------|-----|
| sollicitatiegesprek | `/sɔ.lɪ.si.ˈtaː.tsi.ɣə.sprɛk/` |
| voorbereid | `/ˈvoːr.bə.rɛit/` |
| gepassioneerd | `/ɣə.pɑ.si.oː.ˈneːrt/` |
| uitstekende | `/ˈœyt.steː.kən.də/` |
| eerlijk | `/ˈeːr.lək/` |
