## 1. Morning Routine

| Dutch | English | IPA |
|---|---|---|
| Goedemorgen! | Good morning! | `/ˌɣudəˈmɔrɣə(n)/` |
| Heb je goed geslapen? | Did you sleep well? | `/hɛp jə ɣuːt ɣəˈslaːpə(n)/` |
| Ik ga douchen | I'm going to shower | `/ɪk ɣaː ˈdɑuʃə(n)/` |
| Wat eten we als ontbijt? | What are we having for breakfast? | `/ʋɑt ˈeːtə(n) ʋə ɑls ɔntˈbɛit/` |
| Ik ben laat! | I'm late! | `/ɪk bɛn laːt/` |
| Wil je koffie? | Do you want coffee? | `/ʋɪl jə ˈkɔfi/` |
| Ik moet me aankleden | I need to get dressed | `/ɪk muːt mə ˈaːnkleːdə(n)/` |
| Schiet op! | Hurry up! | `/sxiːt ɔp/` |

## 4. Socializing

| Dutch | English | IPA |
|---|---|---|
| Heb je zin om...? | Do you feel like...? | `/hɛp jə zɪn ɔm/` |
| Zullen we afspreken? | Shall we meet up? | `/ˈzʏlə(n) ʋə ˈɑfspreːkə(n)/` |
| Wat leuk! | How nice! / How fun! | `/ʋɑt løːk/` |
| Gefeliciteerd! | Congratulations! | `/ɣəfeːlisiˈteːrt/` |
| Veel plezier! | Have fun! | `/veːl plɛˈziːr/` |
| Ik verheug me erop | I'm looking forward to it | `/ɪk vərˈhøːx mə ɛrˈɔp/` |
| Heb je zin in een biertje? | Do you fancy a beer? | `/hɛp jə zɪn ɪn ən ˈbiːrtjə/` |
| Dat was gezellig | That was a good time | `/dɑt ʋɑs ɣəˈzɛləx/` |
| Zullen we iets gaan drinken? | Shall we go for a drink? | `/ˈzʏlə(n) ʋə iːts ɣaːn ˈdrɪŋkə(n)/` |
| Kom je ook? | Are you coming too? | `/kɔm jə oːk/` |

## 5. On the Phone

| Dutch | English | IPA |
|---|---|---|
| Met... (naam) | This is... (name) | `/mɛt .../` |
| Kan ik ... spreken? | Can I speak to...? | `/kɑn ɪk ... ˈspreːkə(n)/` |
| Ik bel je later terug | I'll call you back later | `/ɪk bɛl jə ˈlaːtər təˈrʏx/` |
| Kun je dat herhalen? | Can you repeat that? | `/kʏn jə dɑt hɛrˈhaːlə(n)/` |
| Je valt weg | You're breaking up | `/jə vɑlt ʋɛx/` |
| Ik stuur je een berichtje | I'll send you a message | `/ɪk styːr jə ən bəˈrɪxtjə/` |

## 6. Weather Talk

| Dutch | English | IPA |
|---|---|---|
| Wat voor weer wordt het? | What will the weather be like? | `/ʋɑt voːr ʋeːr ʋɔrt ɦɛt/` |
| Het regent | It's raining | `/ɦɛt ˈreːɣənt/` |
| Wat een mooi weer! | What beautiful weather! | `/ʋɑt ən moːi ʋeːr/` |
| Het is koud | It's cold | `/ɦɛt ɪs kɑut/` |
| Het is warm | It's warm | `/ɦɛt ɪs ʋɑrm/` |
| Het waait hard | It's very windy | `/ɦɛt ʋaːit hɑrt/` |
