# Dutch Learning Materials

A comprehensive, beginner-friendly reference for learning Dutch (*Nederlands*). Work through these materials progressively to build your vocabulary, pronunciation, grammar, and conversational ability.

## About Dutch

- **Spoken by**: ~25 million native speakers
- **Where**: The Netherlands, Belgium (Flanders), Suriname, and the Caribbean islands of Aruba, Curaçao, and Sint Maarten
- **Language family**: West Germanic — closely related to English and German
- **Good news for English speakers**: Dutch shares a huge amount of vocabulary and grammar with English, making it one of the easiest languages for English speakers to learn

## How to Use These Materials

Each file is self-contained and readable on its own, but the suggested path below gives you the smoothest progression. Every vocabulary file uses a consistent table format with pronunciation hints and example sentences.

The `data/vocabulary.json` file contains all vocabulary in machine-readable format, suitable for flashcard apps, quiz generators, or Anki import.
